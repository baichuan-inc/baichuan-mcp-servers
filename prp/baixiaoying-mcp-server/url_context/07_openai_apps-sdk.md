# Apps SDK - OpenAI Developer Documentation

> Source: https://developers.openai.com/apps-sdk/

## Overview

**Apps SDK** is OpenAI's framework for building applications that extend ChatGPT's functionality. The documentation covers the complete development lifecycle from planning through deployment.

## Key Sections

### Getting Started

- Quickstart guide
- "What makes a great ChatGPT app?" resource
- App submission guidelines (currently open for applications)

### Development Workflow

The platform organizes creation into three phases:

1. **Plan** - Research use cases and define tools
2. **Build** - Construct MCP servers and ChatGPT UI components
3. **Deploy** - Launch and connect applications

### Core Technical Concepts

- MCP Server architecture
- User experience principles
- UI design guidelines
- Authentication and state management
- Monetization options

### Quality & Standards

The documentation emphasizes submission requirements through multiple guides:

- "ChatGPT app submission guidelines" for quality and policy standards
- UX principles ensuring intuitive interfaces
- UI guidelines for native ChatGPT integration

### Supporting Resources

- Security and privacy considerations
- Metadata optimization for discoverability
- Troubleshooting documentation
- Reference materials

The framework positions itself as comprehensive infrastructure supporting developers in creating commercial ChatGPT extensions while maintaining OpenAI's quality standards.
